# Final Web Development Project

This folder contains a complete multi-page website that meets the listed project requirements.

## Pages

- `index.html` — professional Bootstrap portfolio homepage
- `resume.html` — résumé written in HTML
- `scratch.html` — from-scratch page with custom HTML/CSS
- `app.html` — single-page web app with inline HTML, CSS, and JavaScript

## Supporting files

- `assets/css/main.css` — custom styling for Bootstrap pages
- `assets/css/scratch.css` — custom stylesheet for the scratch page
- `assets/img/` — original local SVG artwork and favicon
- `PROJECT_CHECKLIST.md` — requirement-by-requirement map

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload everything in this folder, preserving the folder structure.
3. Commit the files to the `main` branch.
4. Open the repository’s **Settings**.
5. Select **Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select the `main` branch and the `/ (root)` folder, then save.
8. After GitHub finishes publishing, open the URL shown in the Pages settings.

Your professional homepage will be the root URL. Submit the direct URL ending in `/scratch.html` when the assignment asks for the scratch-page URL.

## Before submitting

Review the résumé wording and replace or remove any details you do not want public. Test the YouTube and Tableau embeds after the website is online. The embeds require an internet connection.
